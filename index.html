This page is about Array and its method in JS

Array is a linear way to store one kind of information. Such as: 


let marks = [11, 22, 33, 44, 55];
console.log(marks);
console.log(marks.length);  // length is a property in array


  • Arrays are object
  • There are key:value pair but little different
  • In arrays we use index instead of key
  • mutable


# Notice:This is not valid to access the index of array: 

 console.log(marks.charAt(2));

To change value:
# Notice: This way is only for array. This is not possible for string. 

marks[0] = 99;
console.log(marks[0]);

Looping over array (try some different by your own such as for of loop and for in lop and look what happens) :

let marks = [99, 22, 33, 44, 55];
for(let i = 0; i < marks.length; i++){
    console.log(marks[i]);
}

push() method to add new value in the end of array: 

let food = ["Burger", "Pizza", "Chips"];
food.push("Noodles", "Sandwich", "French fry");
console.log(food);

pop() method to delete the last element: 

let food = ["Burger", "Pizza", "Chips"];
food.pop();
console.log(food);

toString() method to convert the array temporarily in a string. It converts the number array to a single string: 

let food = ["Burger", "Pizza", "Chips"];
console.log(food.toString());

concat() method to add more then one arrays together. It doesn't change the main array: 

let food = ["chips", "juice"];
let fruit = ["apple", "banana"];
let foodFruit = food.concat(fruit);
console.log(foodFruit);

unshift() method to add a new element in the beginning. It works like push() method.

shift() method to delete the first element of the array. It works like pop() method.

slice() method to get a slice of array like string. It doesn't change the original array.

splice() method to add, replace or delete element in array. It changes the original array:
# to add: 


let num = [1, 2, 3, 4, 5, 6, 7];
num.splice(2, 0, 111);
console.log(num);

#to delete: 

let num = [1, 2, 3, 4, 5, 6, 7];
num.splice(3, 2);
console.log(num);

# to replace: 

let num = [1, 2, 3, 4, 5, 6, 7];
num.splice(1, 1, 777);
console.log(num);

# to delete everything after a specific number: 

let num = [1, 2, 3, 4, 5, 6, 7];
num.splice(3);
console.log(num);
